## 1. 目的

no8(`sim-controller-no8`)を用いて、IPM / IM の **dq軸 2入力 → 2出力(MIMO)** システムの周波数応答(ボード線図)を取得・可視化し、電流制御系および plant の動特性(帯域・位相余裕・dq cross-coupling)を定量評価する機能を提供する。

## 2. スコープ

### 2.1 対象(4象限)

plant 2種 × ループ2種 = **4象限すべて**を対象とする。

| | 開ループ(plant) | 閉ループ(制御系) |
|---|---|---|
| **IPM** | 電圧指令 → 電流(アドミタンス $Y_{dq}$) | 電流指令 → 電流(追従 $T_{dq}$) |
| **IM** | 電圧指令 → 電流(アドミタンス $Y_{dq}$) | 電流指令 → 電流(追従 $T_{dq}$) |

### 2.2 スコープ外(Non-Goals)

- 実機 / HIL への適用 — **`motor_simulation_base` のシミュレーション内に限定**
- 制御器(PIゲイン等)の自動設計・最適化 — **評価のみ**(設計は対象外)
- 安定判別の自動化(一般化 Nyquist / determinant 判定) — 周波数応答の取得・可視化までを範囲とし、安定余裕は読み取り値として提示
- 非線形・大振幅特性(高調波歪み等) — **小振幅線形化前提**

### 2.3 評価対象レイヤと実装経路(2026-06-20 確定)

**「no8」は2つの意味を区別する**: 広義 = `motor_simulation_base` のシミュレーション基盤全体、狭義 = `sim-controller-no8`(実機相当の C 制御コード)。

| レイヤ | 実体 | 実装場所 |
|---|---|---|
| **plant** | IPM / IM 物理モデル | `sim-core`(Rust)`motor_model/ipm.rs`, `im.rs` |
| **制御器(検証・比較用)** | Rust PI | `sim-core`(Rust)`current_pi.rs` |
| **制御器(本命)** | 実機相当 phase7 電流制御 | **狭義 no8**(C)`sim-controller-no8` |

**実装経路**(機能スコープとして確定、詳細は設計フェーズ OI-5):

- **共通経路(開・閉ループ共通、2026-06-20 指示)**: 電圧指令 $V_d/V_q$ 以降から plant への経路は開ループ・閉ループで**共通**とし、いずれも **sim-no8 を通す**構成とする。これにより開・閉の FRF が同一信号経路を通り、クロスチェック(AC-4)が経路差なく成立する。$V_d/V_q$ の注入点・経路の精密な分界は設計フェーズ(OI-5 / OI-6)。
- **開ループ plant アドミタンス $Y_{dq}$** → 制御器(PI)をバイパスし、$V_d/V_q$ を加振信号として直接注入。$V_d/V_q$ 以降は上記共通経路(sim-no8 経由)で plant へ。
- **閉ループ追従 $T_{dq}$** → 評価する制御器が $V_d/V_q$ を生成。制御器を分岐(いずれも $V_d/V_q$ 以降は開ループと同一の共通経路):
  - **Rust PI**(検証・比較用)
  - **狭義 no8 の C 制御**(本命・実機相当)→ sim-no8 ブリッジ経由

## 3. 対象システムの定義

すべて dq 2×2 MIMO に帰着する:

$$
\begin{bmatrix} y_d \\ y_q \end{bmatrix} = G(s)\begin{bmatrix} u_d \\ u_q \end{bmatrix},\quad
G(s)=\begin{bmatrix} G_{dd} & G_{dq} \\ G_{qd} & G_{qq} \end{bmatrix}
$$

- **開ループ**: $u=$ 電圧指令 $(v_d,v_q)$、$y=$ 電流 $(i_d,i_q)$ → $G=$ アドミタンス $Y_{dq}$
- **閉ループ**: $u=$ 電流指令 $(i_d^\*,i_q^\*)$、$y=$ 電流 $(i_d,i_q)$ → $G=$ 追従行列 $T_{dq}$
- 非対角 $G_{dq},G_{qd}$ = **dq cross-coupling**(IPM 突極 $L_d\neq L_q$ / IM で増大)

## 4. 機能要件(FR)

|---|---|---|

| FR-3 | 入力を電圧指令(開)/ 電流指令(閉)として加振できる | MUST |

| FR-5 | 加振方式 **stepped sine** をサポート | MUST |
| FR-6 | 加振方式 **chirp / multisine** をサポート | SHOULD |

| FR-10 | **出力B**: 特異値ボード線図($\bar\sigma,\underline\sigma$) | MUST |
| FR-11 | **出力C**: 複素ベクトル両側ボード線図($G_+,G_-$、$\pm\omega$) | SHOULD |

| FR-13 | **FRF 数値データ**(複素 2×2 vs 周波数)を保存(再描画・後処理用) | MUST |
| FR-14 | 図のエクスポート(png 等) | SHOULD |
| FR-15 | 既知 plant の**理論伝達関数を重ね描き**(検証用) | SHOULD |

## 5. 入出力仕様

### 5.1 入力(評価条件)
- plant 種別: IPM / IM
- ループ: open / closed
- 動作点: 回転速度 $\omega$、電流バイアス $(i_{d0},i_{q0})$
- 周波数: $f_{\min}$, $f_{\max}$, 点数 / decade(対数)
- 加振: 方式(stepped sine / chirp / multisine)、振幅(小振幅)、d/q 個別加振の制御

### 5.2 出力
- **FRF**: 周波数ごとの複素 2×2 $(G_{dd}\dots G_{qq})$
- **派生**: 特異値 $\bar\sigma/\underline\sigma$、複素ベクトル $G_+/G_-$、(開ループ)$Y_{dq}$
- **図**: ボード線図(A)、特異値(B)、両側(C)
- **メタ**: 評価条件・加振パラメータ(再現性確保)

## 6. 非機能要件(NFR)

| ID | 要件 |
|---|---|
| NFR-1 | 動作点の worst-case は**機械 1000rpm 以上**($\omega_e\geq 419$ rad/s、4極対)を最優先前提とする(プロジェクト前提) |
| NFR-2 | dq・電力は **power-invariant** 定義で統一($T_e=p(\phi_d i_q-\phi_q i_d)$、$P=v_d i_d+v_q i_q$) |
| NFR-3 | 加振は線形応答が保たれる**小振幅**。線形性(振幅依存の有無)は別途確認 |
| NFR-4 | 取得 FRF は既知 plant の理論伝達関数と所定誤差内で一致(§7) |
| NFR-5 | 注入点と取得点の**物理量整合**(`v_cmd ↔ i_true` 等)を保証(Phase 0 校正方針と接続) |

## 7. 受け入れ基準(Acceptance Criteria)

- **AC-1**: 既知パラメータの線形 plant に対し、取得 FRF(対角)が解析的伝達関数とゲイン誤差・位相誤差 所定内で一致(許容値は OI-2)。
- **AC-2**: 非対角(cross-coupling)が、動作点・突極比から期待される結合と整合(符号・概形)。
- **AC-3**: IPM の突極 / IM の非対称により **mirror結合 $G_-$** が現れることを両側ボード線図で確認できる。
- **AC-4**: 開ループ $Y_{dq}$ と閉ループ追従 $T_{dq}$ が、制御則を介して整合(クロスチェック)。

## 8. 前提・制約

- no8 シミュレーション内。power-invariant dq。
- 小振幅線形化。**動作点固定**での局所 FRF(動作点ごとに1セット)。
