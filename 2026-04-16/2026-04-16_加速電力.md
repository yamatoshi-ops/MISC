いいポイントです。ここを曖昧にすると補償が破綻します。

結論から言うと、
**加減速時の銅損変化 ΔP_c は「電流の変化」と「抵抗の変化」によって決まります」**。
ただし、時間スケールで支配因子が分かれます。

---

# 1. 基本式（ここからスタート）

三相の銅損は

[
P_c = 3 R_s (i_d^2 + i_q^2)
]

です。

したがって変化量は

[
\Delta P_c
==========

3 R_s \bigl[(i_d^2 + i_q^2) - (i_{d0}^2 + i_{q0}^2)\bigr]
+
3 (R_s - R_{s0})(i_{d0}^2 + i_{q0}^2)
]

です。

---

# 2. 分解すると意味がはっきりする

[
\Delta P_c =
\underbrace{3 R_s \Delta(i^2)}*{\text{電流変化起因}}
+
\underbrace{3 \Delta R_s \cdot i^2}*{\text{温度起因}}
]

ここで

* (i^2 = i_d^2 + i_q^2)

---

# 3. 加減速時に効いているのはどっちか？

## ■ 加減速の時間スケール（ms〜100ms）

👉 **支配的なのは電流変化**

つまり

[
\Delta P_c \approx 3 R_s \Delta(i^2)
]

です。

理由：

* 温度（Rs）はほぼ変わらない（熱時定数は秒〜分）
* 電流は大きく変化する（トルク応答）

---

## ■ 温度ドリフト（秒〜分）

👉 **支配的なのは抵抗変化**

[
\Delta P_c \approx 3 \Delta R_s \cdot i^2
]

これがあなたの補償対象です。

---

# 4. もう少し踏み込む（微分形）

制御的に理解するなら、時間微分形が有効です。

[
P_c = 3R_s(i_d^2 + i_q^2)
]

微分すると

[
\frac{dP_c}{dt}
===============

3R_s \cdot 2(i_d \dot{i}_d + i_q \dot{i}_q)
+
3 \dot{R}_s (i_d^2 + i_q^2)
]

---

## 分解

### ① 電流変化項（過渡支配）

[
6R_s(i_d \dot{i}_d + i_q \dot{i}_q)
]

👉 加減速で支配的

---

### ② 抵抗変化項（温度）

[
3 \dot{R}_s (i_d^2 + i_q^2)
]

👉 ゆっくり変化

---

# 5. 重要な直感

加減速時に何が起きているか：

### ① iq増加 → トルク増加

[
i_q ↑ → i^2 ↑ → P_c ↑
]

### ② 電流応答中

[
\dot{i} ≠ 0 → 瞬間的にP_c変動
]

### ③ id変化（MTPA・FW）

[
i_dも変わる → P_cさらに変動
]

---

# 6. あなたの ΔP にどう混ざるか

あなたの観測電力

[
P_{\text{obs}} = V_d I_d + V_q I_q

[
P_{\text{obs}} = P_c + P_{\text{mag}} + P_{\text{acc}} + ...

が含まれます。

その中の銅損部分が

[
P_c = 3R_s(i_d^2 + i_q^2)

## つまり加減速時の ΔP の内訳

[
\Delta P =
\underbrace{3R_s\Delta(i^2)}*{\text{銅損（速い）}}
+
\underbrace{J\omega\dot{\omega}}*{\text{加速電力}}
+
\underbrace{\frac{dW_{mag}}{dt}}*{\text{磁気エネルギー}}

\underbrace{3\Delta R_s i^2}*{\text{温度（遅い）}}

# 7. ここが重要（あなたの設計との関係）

# 9. なぜ加速電力補正が効くか

結果：

👉 残るのは

👉 上2つを抑える

## ■ 温度起因（補償対象）

はい。
ここはかなり重要で、**ΔPの中に何が混ざっているか** を dq 式からきちんと分解すると、今やっている

* 重み付け
* 積分停止
* 加速電力補正

の意味がかなり明確になります。

まず、添付論文の dq モデルは

[
v_d = R_s i_d + \frac{d}{dt}(L_d i_d) - \omega_e L_q i_q
]

[
v_q = R_s i_q + \frac{d}{dt}(L_q i_q) + \omega_e L_d i_d + \omega_e \Phi
]

で、トルクは

[
\tau_e = N_p(2L_m i_d + \Phi)i_q
]

です。これは添付論文の dq 回路方程式・トルク発生式と一致しています。

以下、この式から電力を完全分解します。

---

# 1. dq モデルから電力式を作る

電力を

[
P_{dq} = v_d i_d + v_q i_q
]

と置きます。
上の dq 方程式を代入すると、

[
P_{dq}
======

R_s i_d^2 + i_d\frac{d}{dt}(L_d i_d) - \omega_e L_q i_d i_q
+
R_s i_q^2 + i_q\frac{d}{dt}(L_q i_q) + \omega_e L_d i_d i_q + \omega_e \Phi i_q
]

整理すると、

[
P_{dq}
======

R_s(i_d^2+i_q^2)
+
i_d\frac{d}{dt}(L_d i_d)
+
i_q\frac{d}{dt}(L_q i_q)
+
\omega_e (L_d-L_q)i_d i_q
+
\omega_e \Phi i_q
]

です。

ここで (L_d-L_q=2L_m) なので、

[
P_{dq}
======

R_s(i_d^2+i_q^2)
+
i_d\frac{d}{dt}(L_d i_d)
+
i_q\frac{d}{dt}(L_q i_q)
+
2\omega_e L_m i_d i_q
+
\omega_e \Phi i_q
]

となります。

---

# 2. トルク項にまとめる

添付論文のトルク式は

[
\tau_e = N_p(2L_m i_d + \Phi)i_q
]

なので、

[
\omega_m \tau_e
===============

\omega_m N_p(2L_m i_d + \Phi)i_q
]

電気角速度と機械角速度の関係は

[
\omega_e = N_p \omega_m
]

だから、

[
\omega_m \tau_e
===============

# \omega_e (2L_m i_d + \Phi)i_q

2\omega_e L_m i_d i_q + \omega_e \Phi i_q
]

となります。

したがって dq 電力は

[
\boxed{
P_{dq}
======

R_s(i_d^2+i_q^2)
+
i_d\frac{d}{dt}(L_d i_d)
+
i_q\frac{d}{dt}(L_q i_q)
+
\omega_m \tau_e
}
]

です。

これがまず一番大事な式です。

---

# 3. この式の意味

この式は

[
\boxed{
P_{dq}
======

P_{cu}
+
P_{mag}
+
P_{mech}
}
]

と読めます。

それぞれ

[
P_{cu}=R_s(i_d^2+i_q^2)
]

[
P_{mag}=i_d\frac{d}{dt}(L_d i_d)+i_q\frac{d}{dt}(L_q i_q)
]

[
P_{mech}=\omega_m\tau_e
]

です。

つまり、あなたが見ている (V_d I_d+V_q I_q) には少なくとも

* 銅損
* 磁気エネルギー変化
* 機械出力

が混ざっています。

---

# 4. 磁気エネルギー項の具体式

ここが質問の本体です。

## 4-1. 一般形

[
P_{mag}=i_d\frac{d}{dt}(L_d i_d)+i_q\frac{d}{dt}(L_q i_q)
]

です。

これを展開すると

[
\frac{d}{dt}(L_d i_d)=L_d\dot{i}_d+i_d\dot{L}_d
]

[
\frac{d}{dt}(L_q i_q)=L_q\dot{i}_q+i_q\dot{L}_q
]

なので

[
\boxed{
P_{mag}
=======

L_d i_d \dot{i}_d
+
L_q i_q \dot{i}_q
+
i_d^2 \dot{L}_d
+
i_q^2 \dot{L}_q
}
]

です。

---

## 4-2. インダクタンス一定近似

まず制御設計ではここから始めるのが普通です。

[
\dot{L}_d \approx 0,\qquad \dot{L}_q \approx 0
]

とすれば

[
\boxed{
P_{mag}
\approx
L_d i_d \dot{i}_d + L_q i_q \dot{i}_q
}
]

です。

これは

* 電流を増やしているとき正
* 電流を減らしているとき負

になるので、**コイル磁気エネルギーの蓄積・放出** を表しています。

---

## 4-3. エネルギーの時間微分として書く

インダクタンス一定なら磁気エネルギーは

[
W_{mag}=\frac{1}{2}L_d i_d^2+\frac{1}{2}L_q i_q^2
]

なので

[
\frac{dW_{mag}}{dt}
===================

L_d i_d \dot{i}_d + L_q i_q \dot{i}_q
]

です。

したがって

[
\boxed{
P_{mag}\approx \frac{dW_{mag}}{dt}
}
]

です。

これが「磁気エネルギー項」の一番わかりやすい形です。

---

# 5. dq モデルから“完全分解した電力式”

ここからさらに機械系も分けます。

機械系は

[
J\dot{\omega}_m = \tau_e - \tau_L - B\omega_m
]

とすると、両辺に (\omega_m) を掛けて

[
\omega_m\tau_e
==============

J\omega_m\dot{\omega}_m + \tau_L\omega_m + B\omega_m^2
]

です。

したがって dq 電力式は

[
P_{dq}
======

P_{cu}
+
P_{mag}
+
J\omega_m\dot{\omega}_m
+
\tau_L\omega_m
+
B\omega_m^2
]

です。

つまり

[
\boxed{
P_{dq}
======

R_s(i_d^2+i_q^2)
+
\frac{dW_{mag}}{dt}
+
J\omega_m\dot{\omega}_m
+
\tau_L\omega_m
+
B\omega_m^2
}
]

となります。

これが、あなたの議論に最も使いやすい“完全分解形”です。

# 6. あなたの ΔP の中に何が混ざるか

いまの観測電力を

[
P_{obs}=V_d I_d+V_q I_q

\Delta P

\Delta P_{cu}

\Delta P_{mag}

\Delta P_{acc}
+

それぞれ

[

[
\Delta P_{acc} \sim \Delta{J\omega_m\dot{\omega}_m}

[
\Delta P_{load} \sim \Delta{\tau_L\omega_m}
]

[
\Delta P_{fric} \sim \Delta{B\omega_m^2}

です。

# 7. 今回の制御課題との対応

あなたが本当に補償したいのは主に

[
\Delta P_{cu,;temp}
\approx \Delta R_s \cdot (i_d^2+i_q^2)
]

のような**温度起因の遅い成分**です。

一方、過渡時に大きく乗るのは

[
\Delta P_{mag} \approx \frac{dW_{mag}}{dt}
]

[
\Delta P_{acc} \approx J\omega_m\dot{\omega}_m
]

です。

だから、今回の施策は理屈として

* (J\omega\dot{\omega}) をモデルで引く
* (\dot{T},\dot{\omega}) ベースの重みで (dW_{mag}/dt) や急変銅損を抑える
* 温度起因の低速成分だけ通す

という構造になります。

この整理はかなり筋が通っています。

---

# 8. 実装しやすい近似式

実際の制御器に載せるなら、まずはこの近似で十分です。

## 観測電力

[
P_{obs}=V_dI_d+V_qI_q
]

## 加速電力

[
P_{acc}=K_{acc}J\omega_m\dot{\omega}_m
]

## 磁気エネルギー項の近似

[
P_{mag,est}=K_{mag}(L_d i_d \dot{i}_d + L_q i_q \dot{i}_q)
]

## 補正後偏差

[
\Delta P_{corr}
===============

## P_{ref}

\left(P_{obs}-P_{acc}-P_{mag,est}\right)
]

あるいは符号定義に合わせて

[
\Delta P_{corr}
===============

\Delta P - P_{acc} - P_{mag,est}
]

です。

ここで (\dot{i}_d,\dot{i}_q) はノイズが強いので、そのままではなく

[
\dot{i}_{d,f}=\mathrm{LPF}\left(\frac{i_d[k]-i_d[k-1]}{T_s}\right)
]

[
\dot{i}_{q,f}=\mathrm{LPF}\left(\frac{i_q[k]-i_q[k-1]}{T_s}\right)
]

を使うのが現実的です。
